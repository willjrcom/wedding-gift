
# Pix Casamento (Next.js + Tailwind)

- Projeto estático (sem backend)
- Link do casal em `?d=...` (JSON encodeado)
- Lista de presentes é igual para todos os casais (app/data/gifts.json)
- Pix aparece **somente** após clicar em "Presentear" na página do produto
- Imagens aceitam qualquer URL (usa `<img>` nativo)

## Rodar
```bash
npm install
npm run dev
```

## Build estático
```bash
npm run build
```
